

#### 1. Running the Script
```bash
./create_cpp_project.sh hello
```

#### 2. Output Structure
```
hello/
├── build/
├── CMakeLists.txt
└── main.cpp
```

#### 3. Building the Project
```bash
cd hello/build
cmake ..
make
```

#### 4. Running the Program
```bash
./hello
```
**Output**:
```
Hello, hello!
```
