

#### 1. Running the Script
```bash
./create_cpp_project.sh Hell
```
#### 2. Output Structure
```bash
Hell/
├── build/
├── CMakeLists.txt
└── main.cpp
```

#### 3. Building the Project
```bash
cd Hell/build
cmake ..
make
```

#### 4. Running the Program
```bash
./Hell
```
**Output**:
```
Hello, Hell!
```
